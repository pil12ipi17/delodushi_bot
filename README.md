# DeloDushi Bot

Telegram-бот для персонализированных нумерологических разборов с webhook-доставкой, оплатой цифровых продуктов, админ-инструментами и управлением контентом через Google Sheets.

## Что Делает Проект

- рассчитывает нумерологические значения по дате рождения пользователя
- отправляет бесплатный разбор прямо в Telegram
- принимает оплату за полный разбор и дополнительные продукты
- автоматически доставляет оплаченный контент после подтверждения платежа
- управляет текстами, продуктами и офферами через Google Sheets
- поддерживает админ-статистику, рассылки и заявки на консультацию

## Технологии

- Python
- Flask
- pyTelegramBotAPI
- gspread
- Google service account credentials
- YooKassa API
- Requests

## Архитектура

- `app.py` содержит Telegram-обработчики, webhook endpoints, платежный поток и логику отложенных офферов
- `config.py` читает runtime-конфигурацию из переменных окружения или локального игнорируемого файла
- Google Sheets используется как легковесная CMS и операционное хранилище для текстов, пользователей, продуктов и офферов
- Flask публикует webhook endpoints для Telegram и YooKassa

## Основной Пользовательский Сценарий

1. Пользователь запускает бота и проходит проверку подписки на канал.
2. Бот запрашивает дату рождения и отправляет короткий бесплатный разбор.
3. Пользователь оплачивает полный разбор или дополнительный продукт.
4. YooKassa отправляет webhook-событие во Flask-приложение.
5. Бот доставляет оплаченный контент и планирует последующие офферы.

## Локальный Запуск

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

Создай один из вариантов:

- `config_local.py` для локальной разработки
- переменные окружения по образцу из `.env.example`

Запуск:

```bash
python app.py
```

## Конфигурация

Ожидаемые настройки:

- `BOT_TOKEN`
- `ADMIN_IDS`
- `SHOP_ID`
- `PAYMENT_TOKEN`
- `GOOGLE_SHEET_NAME`
